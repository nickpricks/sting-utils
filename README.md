# sting-utils

STRING (or in some cases INTEGER/FLOATS) based Utilities

# Current Features

* parseEmptySting
* formatAsCurrency
* capitalizeFirstLetter
* capitalizeWord
* capitalizeFirstLetterInWholeSentence
* getFileExtension = fn => fn.substring(fn.lastIndexOf('.'), fn.length);
* uuid - dependent on *uuid*
* parseIntOrZero
* parseFloatOrZero
* parseFloatToFixed
* prefixZero
* inWords
